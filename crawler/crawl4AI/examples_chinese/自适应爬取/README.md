# 自适应爬取示例

本目录包含演示Crawl4AI自适应爬取功能各个方面的示例。

## 示例概述

### 1. `基本用法.py`
- 自适应爬取的简单介绍
- 使用默认的统计策略
- 展示如何获取爬取统计数据和相关内容

### 2. `嵌入策略.py` ⭐ 新增
- 演示基于嵌入的语义理解策略
- 展示查询扩展和不相关内容检测
- 包括本地和基于API的嵌入配置

### 3. `嵌入与统计对比.py` ⭐ 新增
- 统计策略与嵌入策略的直接对比
- 帮助您为您的用例选择合适的策略
- 展示性能和准确性的权衡

### 4. `嵌入配置.py` ⭐ 新增
- 嵌入策略的高级配置选项
- 不同场景的参数调优指南
- 研究、探索和质量导向爬取的示例

### 5. `高级配置.py`
- 展示两种策略的各种配置选项
- 演示阈值调整和性能优化

### 6. `自定义策略.py`
- 如何实现自己的爬取策略
- 扩展基础CrawlStrategy类
- 针对特殊需求的高级用例

### 7. `知识库导出导入.py`
- 将爬取的知识库导出到JSONL
- 从保存的状态导入并继续爬取
- 用于构建持久知识库

## 快速开始

对于您的第一次自适应爬取体验，请运行：

```bash
python 基本用法.py
```

要尝试带有语义理解的新嵌入策略：

```bash
python 嵌入策略.py
```

要比较策略并查看哪种策略最适合您的用例：

```bash
python 嵌入与统计对比.py
```

## 策略选择指南

### 当以下情况使用统计策略（默认）：
- 处理技术文档
- 查询包含特定术语或代码
- 速度至关重要
- 无法访问API

### 当以下情况使用嵌入策略：
- 查询是概念性的或模糊的
- 需要超越精确匹配的语义理解
- 想要检测不相关内容
- 处理多样化的内容源

## 要求

- 已安装Crawl4AI
- 对于使用本地模型的嵌入策略：`sentence-transformers`
- 对于使用OpenAI的嵌入策略：设置`OPENAI_API_KEY`环境变量
