# 将examples_chinese文件夹下所有Python文件注释转换为中英双语

## 任务概述
将`examples_chinese`文件夹下所有层级的Python文件中的注释转换为中英双语版本，保留原有中文注释，在其后方/下方补充准确的英文翻译。

## 实现步骤

1. **遍历所有Python文件**
   - 使用Glob工具获取所有`.py`文件路径
   - 遍历每个文件进行处理

2. **分析文件内容和注释**
   - 读取每个文件的完整内容
   - 识别单行注释（`#`开头）
   - 识别多行注释/文档字符串（`""" """`或`''' '''`包裹）

3. **处理注释翻译**
   - 对于单行注释：
     - 检查是否已经是中英双语
     - 如果不是，将中文注释翻译为英文，格式为：`# 中文注释  # English comment`
   - 对于多行注释/文档字符串：
     - 检查是否已经是中英双语
     - 如果不是，将中文注释翻译为英文，格式为：`"""中文注释\nEnglish comment"""`

4. **保持代码格式不变**
   - 严格保持代码逻辑、缩进、命名、执行功能完全不变
   - 仅修改注释内容
   - 确保翻译后的注释符合Python代码注释的简洁风格

5. **保存修改后的文件**
   - 将处理后的内容写回原文件

## 注意事项

- 确保专业术语（如函数名、参数、技术概念）翻译准确
- 处理已经有中英双语的注释时，跳过或保持原样
- 注意处理不同格式的注释，包括行内注释和块注释
- 确保文件编码和换行符保持不变

## 预期结果

所有Python文件中的注释都将转换为中英双语版本，保留原有中文注释，同时添加准确的英文翻译，代码功能和格式保持不变。
