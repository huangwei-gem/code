# MCP (模型上下文协议) 完全指南

欢迎使用MCP完全指南！本教程系列旨在帮助您从零开始学习和掌握模型上下文协议(Model Context Protocol, MCP)，一个为大语言模型提供上下文和工具的标准化协议。

## 内容概览

本教程分为以下几个部分：

### 1. 基础篇 (初级用户)

- [1.1 MCP简介](1.1_MCP简介.md) - MCP是什么及其解决的问题，核心概念和工作原理 [→相关实现](../mcp_中文.md)
- [1.2 环境搭建](1.2_环境搭建.md) - 安装MCP库及依赖，配置开发环境，使用MCP CLI工具 [→环境配置](../../.env)
- [1.3 第一个MCP服务](1.3_第一个MCP服务.md) - 创建简单的MCP服务器，定义基本资源和工具 [→示例服务器](../../example/example_server.py)
- [1.4 客户端基础](1.4_客户端基础.md) - 创建MCP客户端，连接到服务器，执行基本操作 [→示例客户端](../../example/example_client.py)

### 2. 进阶篇 (中级用户)

- [2.1 资源深度解析](2.1_资源深度解析.md) - 资源类型和URI模式，创建静态和动态资源 [→资源示例](../../mcp/server)
- [2.2 工具高级用法](2.2_工具高级用法.md) - 工具设计最佳实践，参数类型和验证，异步工具函数 [→工具示例](../../example/calculate.py)
- [2.3 提示模板系统](2.3_提示模板系统.md) - 提示设计原则和模式，参数化提示模板，多轮对话提示 [→提示示例](../../mcp.md)
- [2.4 客户端高级功能](2.4_客户端高级功能.md) - 错误处理和重试策略，流式数据处理 [→高级客户端](../../mcp/client)
- [2.5 服务器生命周期和上下文](2.5_服务器生命周期和上下文.md) - 服务器启动和关闭流程，上下文对象深入理解 [→服务器实现](../../server_sse.py)

### 3. 高级篇 (高级用户)

- [3.1 定制传输层](3.1_定制传输层.md) - MCP传输协议详解，创建自定义传输实现 [→传输层实现](../../example/sse)

### 4. 实战项目

## 使用指南

本教程采用循序渐进的方式，建议按顺序阅读和实践。对于有特定需求的读者，也可以直接跳转到相关章节。每个章节都包含理论讲解和实践示例，建议边学习边动手实践。

## 先决条件

- 基本的Python编程知识
- 对异步编程有基本了解
- 熟悉命令行操作
- 了解大语言模型的基本概念
