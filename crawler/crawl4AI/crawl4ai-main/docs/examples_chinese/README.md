# Crawl4AI 中文示例代码

这个文件夹包含了 Crawl4AI 项目的中文示例代码，帮助中文用户更好地理解和使用这个强大的网络爬虫工具。

## 🚀 快速开始

如果你是第一次使用 Crawl4AI，请先阅读 [快速开始指南](快速开始指南.md)。

## 📚 示例代码列表

### 🎯 基础示例（推荐新手从这里开始）
- `hello_world.py` - 最基础的爬虫示例，展示基本概念
- `quickstart.py` - 快速入门指南，包含所有主要功能
- `requirements.txt` - 示例代码所需的依赖包

### ⚙️ 高级功能
- `browser_optimization_example.py` - 浏览器性能优化
- `hooks_example.py` - 钩子函数使用示例
- `进阶示例.md` - 高级功能详细教程

### 🤖 自适应爬虫
- `adaptive_crawling/basic_usage.py` - 自适应爬虫基础使用
- `adaptive_crawling/README.md` - 自适应爬虫详细说明

## 🛠️ 环境准备

### 1. 安装 Crawl4AI
```bash
pip install crawl4ai
```

### 2. 安装浏览器依赖
```bash
crawl4ai-setup
# 或者
playwright install chromium
```

### 3. 安装示例依赖
```bash
pip install -r requirements.txt
```

## 📖 学习路径

### 🔰 新手入门（5-10分钟）
1. **运行 Hello World**: `python hello_world.py`
2. **查看快速入门**: 阅读 [快速开始指南](快速开始指南.md)
3. **运行完整示例**: `python quickstart.py`

### 🚀 进阶学习（30分钟）
1. **浏览器优化**: `python browser_optimization_example.py`
2. **钩子函数**: `python hooks_example.py`
3. **阅读进阶教程**: [进阶示例.md](进阶示例.md)

### 🎯 专业应用（1小时+）
1. **自适应爬虫**: `python adaptive_crawling/basic_usage.py`
2. **查看原始示例**: 参考 `../examples/` 目录
3. **阅读官方文档**: https://crawl4ai.com
