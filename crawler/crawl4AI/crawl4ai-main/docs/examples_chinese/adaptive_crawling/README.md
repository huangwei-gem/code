# 自适应爬虫示例

这个文件夹包含了 Crawl4AI 自适应爬虫功能的中文示例代码。

## 什么是自适应爬虫？

自适应爬虫是 Crawl4AI 的高级功能，它能够：
- 智能分析网页内容的相关性
- 自动决定何时停止爬取
- 根据查询目标动态调整爬取策略
- 建立知识库并评估信息充分性

## 示例文件说明

### 基础示例
- `basic_usage.py` - 最基础的自适应爬虫使用示例

### 高级配置
- `advanced_configuration.py` - 高级配置选项
- `custom_strategies.py` - 自定义策略
- `embedding_configuration.py` - 嵌入策略配置

## 核心概念

### 1. 相关性评分
自适应爬虫会根据你的查询对爬取的内容进行相关性评分。

### 2. 置信度评估
系统会评估已收集信息的充分性，给出置信度分数。

### 3. 爬取策略
- **统计策略**：基于词频统计进行相关性判断
- **嵌入策略**：基于语义理解进行相关性判断

## 使用建议

1. 先运行 `basic_usage.py` 了解基本概念
2. 根据需求选择合适的策略
3. 调整配置参数以优化爬取效果
4. 监控置信度变化，合理设置停止条件

## 注意事项

- 自适应爬虫比普通爬虫消耗更多资源
- 建议设置合理的爬取深度限制
- 注意遵守目标网站的 robots.txt 规则
