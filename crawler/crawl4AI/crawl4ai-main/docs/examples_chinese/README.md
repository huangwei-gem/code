# Crawl4AI 中文示例代码

这个文件夹包含了 Crawl4AI 项目的中文示例代码，帮助中文用户更好地理解和使用这个强大的网络爬虫工具。

## 示例分类

### 基础示例
- `hello_world.py` - 最基础的爬虫示例
- `quickstart.py` - 快速入门指南
- `basic_usage.py` - 基础使用方法

### 高级功能
- `adaptive_crawling/` - 自适应爬虫示例
- `extraction_strategies/` - 数据提取策略示例
- `browser_optimization/` - 浏览器优化示例

### 实际应用
- `amazon_product_extraction/` - 亚马逊商品提取
- `google_search/` - 谷歌搜索爬虫
- `proxy_examples/` - 代理使用示例

## 使用方法

1. 安装 Crawl4AI：
```bash
pip install crawl4ai
```

2. 运行示例代码：
```bash
python hello_world.py
```

## 注意事项

- 所有示例代码都已经翻译成中文注释
- 部分示例可能需要额外的依赖包
- 某些网站可能需要特殊配置才能爬取

## 学习建议

建议按照以下顺序学习：
1. 先运行 `hello_world.py` 了解基本概念
2. 查看 `quickstart.py` 学习常用功能
3. 根据需求查看特定功能的示例
4. 参考 `adaptive_crawling/` 学习高级用法
